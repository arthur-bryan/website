# Experience Template

Copy this HTML block to `content/experience.md` to add a new position.

---

## Timeline Item Template

```html
<div class="timeline-item">
  <span class="timeline-date">Month Year - Present</span>
  <h3 class="timeline-title">Job Title</h3>
  <span class="timeline-company">Company Name</span>
  <div class="timeline-description">
    <ul>
      <li>Achievement or responsibility with quantifiable impact</li>
      <li>Technologies used: AWS, Terraform, Python</li>
      <li>Key project or initiative you led or contributed to</li>
      <li>Collaboration with teams or stakeholders</li>
    </ul>
  </div>
</div>
```

---

## Example with Strong Action Verbs

```html
<div class="timeline-item">
  <span class="timeline-date">January 2024 - Present</span>
  <h3 class="timeline-title">Senior Cloud Engineer</h3>
  <span class="timeline-company">Tech Company Inc.</span>
  <div class="timeline-description">
    <ul>
      <li>Architected and deployed multi-region AWS infrastructure serving 1M+ daily users</li>
      <li>Reduced infrastructure costs by 40% through Reserved Instances and right-sizing</li>
      <li>Automated CI/CD pipelines using GitHub Actions and Terraform, cutting deployment time by 75%</li>
      <li>Led migration of 50+ microservices from on-premises to EKS with zero downtime</li>
      <li>Mentored team of 3 junior engineers on IaC best practices</li>
    </ul>
  </div>
</div>
```

---

## Strong Action Verbs for Tech Roles

**Building/Creating:**
- Architected, Built, Created, Designed, Developed, Engineered, Implemented

**Improving:**
- Automated, Enhanced, Optimized, Reduced, Refactored, Streamlined

**Leading:**
- Coordinated, Led, Managed, Mentored, Orchestrated, Spearheaded

**Analyzing:**
- Analyzed, Audited, Diagnosed, Evaluated, Investigated, Troubleshot

---

## Metrics to Include

- Time saved (reduced by X%)
- Cost savings ($X or X%)
- Scale (X users, X requests/sec, X TB data)
- Reliability (X% uptime, X% reduction in incidents)
- Team impact (mentored X engineers, led team of X)
